## ✨ 功能特点

- 📱 **移动端优先** — 完美适配手机浏览和分享
- 🎨 **精致 UI** — 粉色系设计，圆角卡片，流畅动画
- 🎭 **8 种人设结果** — 低调潜水观察员、精致营业艺术家、间歇性发疯型选手等
- 📊 **四项维度指数** — 表达欲、神秘感、发疯指数、社交电量
- 🔄 **一键重测** — 轻松再玩一次

## 🕹️ 人设类型

| 人设 | 描述 |
|------|------|
| 🔇 低调潜水观察员 | 朋友圈里的隐形观察员，知道一切但很少出现 |
| 🎨 精致营业艺术家 | 不是发朋友圈，是在办一场私人展览 |
| 🌪️ 间歇性发疯型选手 | 平时安静如鸡，突然一晚连发五条 |
| 🎉 朋友圈气氛组组长 | 只要你在线，别人的朋友圈就不会冷场 |
| 🌙 深夜情绪诗人 | 白天正常生活，晚上开始和月亮、宇宙、人生谈判 |
| 📸 人间真实记录仪 | 朋友圈不负责完美，负责证明生活真的发生过 |
| 🎭 神秘高级 NPC | 很少出现，但每次出现都让人脑补一整段故事 |
| 💬 快乐废话制造机 | 朋友圈没有重点，但很有生命力 |

## 🛠️ 技术栈

- **Framework**: Next.js 15
- **UI**: React 19 + Tailwind CSS
- **动画**: CSS Animations
- **语言**: TypeScript

## 🚀 本地开发

```bash
# 克隆仓库
git clone https://github.com/mybookstores/quiz-site.git
cd quiz-site

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打开 http://localhost:3000
```

## 📦 部署

项目可以轻松部署到 Vercel、Netlify 或任何支持 Next.js 的平台：

```bash
# Vercel
npm i -g vercel && vercel

# 或者 GitHub Actions 自动部署
```
